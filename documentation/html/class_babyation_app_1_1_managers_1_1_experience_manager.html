void BabyationApp.Managers.ExperienceManager.Save ( ExperienceModel  experience)

Save changes to a user or preset experience.

Note that this may change the UserExperiences and PresetExperiences list items. In general a new experience will be added to the UserExperiences list, and a preset experience will be moved to the UserExperiences list if it is edited.
